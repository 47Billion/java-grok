How to use Grok

How to use Grok in your project?

Include Java Grok into your java project and use it like:

Grok g = new Grok();
g.addPatternFromFile(/path/to/pattern);
g.compile("%{URI}");
Match gm = g.match(yourlog);
gm.captures();
//See the result
System.out.println(gm.toJson());
